# Jei's Backendlab

A Laravel portfolio site with a dashboard, certifications, contact form, translations, and uploaded certificate images.

## What you need
- PHP 8.3+
- Composer
- Node.js + npm
- MySQL if you want to use a database server, or SQLite for local testing

## Fast setup
1. Copy `.env.example` to `.env`.
2. Generate an app key if it is empty:
   - `php artisan key:generate`
3. Install dependencies:
   - `composer install`
   - `npm install`
4. Prepare storage and database:
   - `php artisan storage:link`
   - `php artisan migrate --seed`
5. Start the app:
   - `php artisan serve`
   - `npm run dev`

## Important .env values
If you only want the project running locally, these are the only values you really need to care about:

- `APP_NAME`: site name
- `APP_ENV`: `local` for your machine, `production` for hosting
- `APP_DEBUG`: `true` locally, `false` in production
- `APP_URL`: your local URL or real domain
- `APP_KEY`: generated by Laravel
- `DB_CONNECTION`: `sqlite` is easiest locally, `mysql` if you are using a remote database
- `SESSION_DRIVER`: `file` is the safest default for shared hosting
- `CACHE_STORE`: `file` if you do not need database cache
- `QUEUE_CONNECTION`: `sync` unless you use queues

## For InfinityFree or shared hosting
- Set `APP_ENV=production`
- Set `APP_DEBUG=false`
- Set `APP_URL` to your real domain
- Use `SESSION_DRIVER=file`
- Use `CACHE_STORE=file`
- Use `QUEUE_CONNECTION=sync`
- Make sure `vendor/` is uploaded
- Make sure `public/` is the web root or that `index.php` points to it correctly
- Clear any old compiled views if you can access the server files: `storage/framework/views`

## Notes
- Certificate images are stored in `storage/app/public/certifications`.
- The certifications form uses `CERT_SECRET` to protect create/delete actions.
- If you do not use AWS, leave the AWS variables empty or remove them from your local `.env`.

## If something breaks
The most common causes are:
- missing `vendor/`
- wrong `APP_URL`
- stale view cache
- wrong document root
- missing `.env` or empty `APP_KEY`
